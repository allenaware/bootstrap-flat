Overview

Bootstrap Flat is a custom theme on top of Bootstrap that provides a flat look to the user interface elements. That means there are no gradients, shadows, or rounded corners. To keep things as simple as possible, all of the CSS which achieves the flat look is contained in the bootstrap-flat.css stylesheet, allowing you to flatten your Bootstrap simply by including one additional CSS file. There are no style changes needed!

In addition to the flattening, a bootstrap-flat-extras.css stylesheet is also available. This stylesheet adds additional (new) modern user interface elements and additional styles for some of the existing Bootstrap elements.

How to use

Using Bootstrap Flat is a simple two-step process and is designed to ensure future upgrades are as easy as possible.

  1. Download and include Bootstrap into your templates.
  2. Download and include Bootstrap Flat into your templates after the Bootstrap CSS.

Install using NuGet

To install Bootstrap Flat, run the following command in the Package Manager Console.

PM> Install-Package Twitter.Bootstrap.Flat -Pre

What's included

Within the download you'll find the following directories and files, logically grouping common assets and providing both compiled and minified variations.

Once downloaded, unzip the compressed folder to see the structure of (the compiled) Bootstrap-flat. You'll see something like this: 

bootstrap/
+-- css/
|   +-- bootstrap-flat.css
|   +-- bootstrap-flat.min.css
|   +-- bootstrap-flat-extras.css
|   +-- bootstrap-flat-extras.min.css

Browser support

Since Bootstrap Flat is built on top of Bootstrap, it has the same browser support as Bootstrap. It is built to work best in the latest desktop and mobile browsers, meaning older and less advanced browsers might receive a less stylized, though fully functional, version of certain components.
