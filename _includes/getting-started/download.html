Install with Bower

You can also install and manage Bootstrap Flat's Less, CSS, JavaScript, and fonts using Bower:

{% highlight bash %}$ bower install bootstrap-flat{% endhighlight %}

Install with npm

You can also install Bootstrap Flat using npm:

{% highlight bash %}$ npm install bootstrap-flat{% endhighlight %}

require('bootstrap-flat') will load all of Bootstrap Flat's jQuery plugins onto the jQuery object. The bootstrap-flat module itself does not export anything. You can manually load Bootstrap Flat's jQuery plugins individually by loading the /js/*.js files under the package's top-level directory.

Bootstrap Flat's package.json contains some additional metadata under the following keys:

  • less - path to Bootstrap Flat's main Less source file
  • style - path to Bootstrap Flat's non-minified CSS that's been precompiled using the default settings (no customization)

Install with Composer

You can also install and manage Bootstrap Flat's Less, CSS, JavaScript, and fonts using Composer:

{% highlight bash %}$ composer require scottdorman/bootstrap-flat{% endhighlight %}

Autoprefixer required for Less/Sass

Bootstrap-flat uses Autoprefixer to deal with CSS vendor prefixes. If you're compiling Bootstrap Flat from its Less source and not using our Gruntfile, you'll need to integrate Autoprefixer into your build process yourself. If you're using precompiled Bootstrap or using our Gruntfile, you don't need to worry about this because Autoprefixer is already integrated into our Gruntfile.
