A flat theme for Bootstrap, inspired by Microsoft's Modern Design Language (originally created for Windows Phone and Windows 8 and updated for Windows 10), Google's Material Design Language, and Apple's flatter design language for iOS.

Download Bootstrap Flat

Currently v3.3.4

Simple customizations.

It's built on top of Bootstrap and uses the recommended method of providing "light" customizations. In other words, these customizations are pure CSS, allowing you to easily upgrade Bootstrap without risk.

Extras features.

Bootstrap Flat can also give you additional CSS styles and components to add common elements like toggle switches, pricing tables, callouts, and more.

Bootstrap Flat is open source. It's hosted, developed, and maintained on GitHub.
