Simple customizations.

It's built on top of Bootstrap and uses the recommended method of providing "light" customizations. In other words, these customizations are pure CSS, allowing you to easily upgrade Bootstrap without risk.

Extra features.

Bootstrap Flat can also give you additional CSS styles and components to add common elements like toggle switches, pricing tables, callouts, and more. Check out the rest of the documentation for more information.

Bootstrap Flat is open source. It's hosted, developed, and maintained on GitHub.
